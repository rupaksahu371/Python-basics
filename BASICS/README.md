# Python Basic Programs

This folder contains fundamental Python programs for beginners to learn and practice basic programming concepts.

## Programs Overview

| # | File Name | Description |
|---|-----------|-------------|
| 1 | `BREAKSTATEMAENT.py` | Demonstrates the use of `break` statement to exit loops |
| 2 | `CONTINUESTATEMENT.py` | Demonstrates the use of `continue` statement to skip iterations |
| 3 | `DIFFERINTORSTRFROMONEINPUT.py` | Accepts input and differentiates between integer and string |
| 4 | `fibbonacciseries.py` | Generates and displays the Fibonacci series |
| 5 | `FINDSUMOFNUMSTOREDINLIST.py` | Calculates the sum of numbers stored in a list |
| 6 | `flower.py` | A program related to flowers (logic-based) |
| 7 | `REVERSEDANYNUMBER.py` | Reverses any given number |
| 8 | `to average of your submarks.py` | Calculates the average of subject marks |
| 9 | `TO CHECK A PERSON IS ELIGIBLE OR NOT FOR VOTING.py` | Checks voting eligibility based on age |
| 10 | `to check a triangle is equilateral,isosceles or scalene.py` | Determines the type of triangle based on sides |
| 11 | `TO CHECK A YEAR IS LEAP OR NOT.py` | Checks if a given year is a leap year |
| 12 | `to check prime numbers.py` | Checks if a number is prime or not |
| 13 | `to check wheather num is odd or even.py` | Determines if a number is odd or even |
| 14 | `TO FIND LIBRARY FINE.py` | Calculates library fine based on days overdue |
| 15 | `to find youngest people.py` | Finds the youngest person among a group |
| 16 | `to insure a person.py` | Insurance-related program |
| 17 | `TO PRINT FIBBONACCI SERIES.py` | Prints Fibonacci series up to a limit |
| 18 | `to reverse any num.py` | Reverses any given number |
| 19 | `TOADDNATURALNUMUPTON.py` | Adds natural numbers up to n |

## Concepts Covered

- **Control Flow**: if-else statements, loops (for, while)
- **Break & Continue**: Loop control statements
- **Functions**: Function definitions and usage
- **Lists**: List operations and manipulations
- **Input/Output**: User input and output display
- **Mathematical Operations**: Basic arithmetic and number operations
- **String Operations**: String manipulations

## How to Run

1. Make sure you have Python installed on your system
2. Navigate to the BASICS folder
3. Run any program using the command:
   
```
bash
   python <filename>.py
   
```
   Example:
   
```
bash
   python to check prime numbers.py
   
```

## Requirements

- Python 3.x or higher
