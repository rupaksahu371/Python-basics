## Getting Started

### Prerequisites
- Python 3.x or higher
- Jupyter Notebook (for data analysis)

### Running Basic Programs
```
bash
cd BASICS
python <filename>.py
```

### Running Projects
```
bash
cd "PYTHON PROJECTS"
python "LOGIN WINDOW/LOGINWINDOWINPYTHON.py"
```
